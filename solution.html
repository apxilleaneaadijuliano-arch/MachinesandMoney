Combine Virtual Tools

Our solution is to use virtual simulation tools together with makerspace access. This way, students and farmers can learn skills without needing to own expensive machines. Real equipment is only used when necessary, saving costs.

Poster

Each virtual tool will be paired with a poster. These posters explain how the tool works and guide learners step by step. This makes learning easier and more accessible, even without owning machines.
